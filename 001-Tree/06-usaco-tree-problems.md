# Lesson 6: USACO Tree Problem List

This file is a small collection of USACO problems related to trees.

Use it as a practice list:

- start from the easier ones
- read the problem carefully
- identify whether it is about traversal, rooting a tree, tree DP, or path queries

## Suggested Starting Set

1. `Silver` - [Cowntagion](https://usaco.org/index.php?page=viewproblem2&cpid=1062)
2. `Silver` - [Clock Tree](https://usaco.org/index.php?page=viewproblem2&cpid=1016)
3. `Silver` - [Barn Tree](https://usaco.org/index.php?page=viewproblem2&cpid=1254)

## How to Use This List

When you open a problem, ask yourself:

- is the input already rooted, or do I need to root the tree myself?
- do I need DFS, BFS, or both?
- is the answer about a subtree, a path, or the whole tree?
- do I need to store `parent_id` and `child_ids`, or is an undirected graph enough?

If you can classify the problem before coding, the implementation becomes much easier.
