# 003 - Breadth First Search

This folder contains the Breadth First Search section of the algorithm course.

## Course Goals

In this module, we will learn:

1. What BFS means and why it explores a graph level by level
2. How to implement BFS safely using a queue and a `visited` structure
3. How BFS helps with connected components and traversal order
4. Why BFS is the standard tool for shortest path in unweighted graphs
5. How grid BFS appears in contest problems
6. How multi-source BFS starts from many states at once
7. How BFS can search state spaces where one state is a whole configuration
8. How BFS shows up in real USACO Bronze and Silver problems
