# 002 - Depth First Search

This folder contains the Depth First Search section of the algorithm course.

## Course Goals

In this module, we will learn:

1. What DFS means and how recursive DFS is structured
2. Why generic DFS often needs a `visited` array
3. How flood fill works on 2D grids
4. How DFS can replace a changing number of nested loops
5. How to solve classic search problems such as permutations and N-Queens
6. How DFS appears in real USACO Bronze and Silver problems

## Lessons

1. [01-dfs-basics.md](./01-dfs-basics.md)
   DFS basics, recursion structure, and why graphs and state spaces need `visited`
2. [02-flood-fill.md](./02-flood-fill.md)
   Flood fill on grids, connected components, and counting area or region size
3. [03-dynamic-layers-of-search.md](./03-dynamic-layers-of-search.md)
   DFS as dynamic nested loops for permutations, subsets, and the 8 queens problem
4. [04-usaco-dfs-problems.md](./04-usaco-dfs-problems.md)
   Six official USACO Silver DFS practice problems from the contest archive

More lessons can be added later as the course grows.
